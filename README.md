A lightweight, beginner-friendly fitness tracking app built using React Native and Expo. The app focuses on simplicity and motivation, giving users an easy way to monitor daily activity and stay engaged with fitness goals.

Core Features:

Live Step Counter – Uses the device pedometer to track your steps in real time.

Progress Tracking – Visual progress bar showing how close you are to your daily step goal.

User Profiles – Users can create and save a custom username.

Quick Navigation – Access exercises, settings, login, and registration from the main dashboard.

Clean UI – Gradient backgrounds, colourful widgets, and friendly design elements.
